# Парсинг продуктов с сайта 1000.menu

## Использование

### Вариант 1: Через npm (рекомендуется) ⭐

Просто выполните команду:

```bash
npm run parse-products
```

Скрипт автоматически:
- Откроет браузер (headless режим)
- Загрузит страницу с таблицей
- Извлечет все продукты
- Сохранит результат в `scripts_parsing/products/products-1000menu.json`

### Вариант 2: Выполнение в браузере

1. Откройте https://1000.menu/food-table в браузере
2. Откройте консоль разработчика (F12 или Cmd+Option+I на Mac)
3. Скопируйте и вставьте код из файла `parse-1000menu.js`
4. Нажмите Enter для выполнения
5. Скрипт автоматически скачает файл `products-1000menu.json` с результатами

### Вариант 3: Прямой запуск Node.js скрипта

```bash
node scripts_parsing/products/parse-1000menu-node.js
```

## Формат выходных данных

Результат будет в формате JSON с массивом продуктов:

```json
[
  {
    "name": "Молоко",
    "description": "Калорийность: 64 ккал на 100г. Белки: 3г, Жиры: 3г, Углеводы: 5г",
    "categories": [],
    "photo": null,
    "macros": {
      "calories": 64,
      "proteins": 3.0,
      "fats": 3.0,
      "carbs": 5.0,
      "isEmpty": false
    },
    "measures": [
      {
        "name": "Миллилитр",
        "weight": 1,
        "isDefault": true,
        "density": 1
      }
    ]
  }
]
```

## Фильтрация

Скрипт автоматически:
- Исключает продукты с брендами (Dr.Körner, Данон, и т.д.)
- Очищает названия от лишних деталей (категории, проценты жирности, и т.д.)
- Удаляет дубликаты
- Определяет базовые меры для продуктов (граммы, миллилитры, штуки)
