# Описание релиза

## Новые фичи

### Поддержка бейджей в админ-панели
- Добавлена полная поддержка управления бейджами в админ-панели
- Создание новых бейджей с полной формой (название, код, описание, тип субъекта, раритетность, иконка, условия получения)
- Редактирование существующих бейджей через модальное окно
- Управление списком бейджей с поиском и фильтрацией
- Интеграция с API `/v1/badges` для CRUD операций

### Сохранение состояния вкладок
- Добавлено автоматическое сохранение активной вкладки и подвкладки в localStorage
- Состояние восстанавливается при перезагрузке страницы
- Реализовано для всех компонентов Tabs в проекте:
  - Админ-панель (`refook_admin`)
  - Холодильник (`refook_fridge`)
  - Профиль пользователя (`refook_profile`)
  - Подвкладки продуктов (`refook_admin_products`)

## Общие изменения

### Рефакторинг админ-панели
- Унификация компонентов для работы с именованными сущностями (теги, категории, кухни, бейджи)
- Создан единый реестр сущностей (`ENTITY_REGISTRY`) для централизованного управления конфигурацией
- Вынесены общие компоненты в директорию `shared/`:
  - `NamedEntitySubTabs` - универсальный компонент для создания и управления сущностями
  - `NamedEntityManager` - компонент для управления списком сущностей с поиском и редактированием
  - `CreateEntityForm` - универсальная форма создания простых сущностей
  - `CreateBadgeForm` - специализированная форма создания бейджей
  - `EditBadgeModal` - модальное окно редактирования бейджей
- Перенесены хуки в общую директорию `hooks/`:
  - `useCopyToClipboard` - для копирования ID в буфер обмена
  - `useProductEditing` - для редактирования продуктов
  - `useVariantEditing` - для редактирования вариантов продуктов
- Удалены дублирующиеся компоненты (TagSubTabs, CategorySubTabs, KitchenSubTabs, TagManager, CategoryManager)
- Упрощена структура компонентов для упрощения добавления новых типов сущностей

### Улучшение компонента Tabs
- Добавлена поддержка опционального параметра `storageKey` для автоматического сохранения состояния
- Реализована синхронизация состояния с внешним компонентом через колбэк `onChange`
- Добавлена поддержка восстановления состояния подвкладок из localStorage
- Улучшена обработка ошибок при работе с localStorage

### Документация
- Добавлена подробная JSDoc документация для всех новых компонентов:
  - `Tabs` - компонент вкладок с полным описанием API и примеров использования
  - `CreateBadgeForm` - форма создания бейджей с описанием всех полей
  - `EditBadgeModal` - модальное окно редактирования с описанием поведения
  - `NamedEntitySubTabs` - универсальный компонент подвкладок
  - `NamedEntityManager` - компонент управления сущностями
- Обновлена документация `ADMIN_SUBTABS_GUIDE.md` с описанием новой архитектуры

### Улучшение профиля пользователя
- Убрана синхронизация вкладок с URL параметрами, теперь используется только localStorage
- Упрощена логика навигации между разделами профиля

## Багфиксы

### Исправление зависания модального окна редактирования бейджей
- Исправлена проблема с зависанием модального окна при сохранении изменений
- Модальное окно теперь закрывается немедленно при нажатии "Сохранить", а сохранение происходит в фоне
- Улучшена обработка ошибок при сохранении (логирование в консоль без блокировки UI)

### Исправление синхронизации состояния вкладок
- Исправлена проблема с синхронизацией активной вкладки при изменении пропса `initial`
- Добавлена корректная обработка случая, когда `storageKey` указан и значение загружается из localStorage
- Исправлена логика восстановления состояния подвкладок при переключении между вкладками
