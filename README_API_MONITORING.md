# Как узнать какой JSON отправляется на API

## 🎯 Цель
Узнать структуру JSON данных, которые отправляются на эндпоинт `http://localhost:5173/api/v1/recipe` при создании рецепта.

## 🔧 Способы мониторинга

### 1. Встроенный API Logger (Рекомендуется)

В правом верхнем углу приложения есть панель **DevTools** с разделом **API Logger**:

1. Откройте приложение
2. Найдите панель DevTools в правом верхнем углу
3. Разверните панель (нажмите на иконку 📁)
4. Найдите раздел **API Logger**
5. Создайте новый рецепт
6. В API Logger вы увидите все запросы к API в реальном времени

**Преимущества:**
- ✅ Работает в реальном времени
- ✅ Показывает полную структуру JSON
- ✅ Можно фильтровать по URL
- ✅ Можно экспортировать логи
- ✅ Сохраняет историю в localStorage

### 2. Консоль браузера

Откройте DevTools (F12) и используйте глобальный объект `apiLogger`:

```javascript
// Получить все логи API
apiLogger.getLogs()

// Получить логи только для рецептов
apiLogger.getLogsForUrl('/api/v1/recipe')

// Получить последний запрос
apiLogger.getLastRequest()

// Экспортировать все логи
apiLogger.exportLogs()

// Очистить логи
apiLogger.clearLogs()
```

### 3. Network панель браузера

1. Откройте DevTools (F12)
2. Перейдите на вкладку **Network**
3. Создайте новый рецепт
4. Найдите запрос к `/api/v1/recipe`
5. Кликните на запрос и перейдите на вкладку **Payload** или **Request**

### 4. localStorage

Логи автоматически сохраняются в localStorage:

```javascript
// Получить все логи из localStorage
const logs = JSON.parse(localStorage.getItem('api_logs') || '[]')
console.log(logs)
```


## 🚀 Быстрый старт

1. **Запустите приложение:**
   ```bash
   npm run dev
   ```

2. **Откройте DevTools:**
   - Нажмите F12 или правой кнопкой → "Inspect"

3. **Найдите API Logger:**
   - В правом верхнем углу найдите панель DevTools
   - Разверните её и перейдите в раздел API Logger

4. **Создайте рецепт:**
   - Перейдите на страницу создания рецепта
   - Заполните форму и отправьте

5. **Просмотрите JSON:**
   - В API Logger вы увидите запрос к `/api/v1/recipe`
   - Кликните на запрос для просмотра деталей

## 🔍 Отладка

Если API Logger не работает:

1. Проверьте консоль на ошибки
2. Убедитесь, что приложение запущено в режиме разработки
3. Попробуйте очистить localStorage: `localStorage.removeItem('api_logs')`
4. Перезагрузите страницу

## 💡 Советы

- API Logger автоматически обновляется каждую секунду
- Можно фильтровать запросы по URL
- Логи сохраняются между сессиями
- Можно экспортировать логи в JSON файл
