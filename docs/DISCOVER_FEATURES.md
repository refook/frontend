# Раздел Discover - Документация

## Обзор

Раздел "Discover" представляет собой интерактивный интерфейс для быстрого просмотра рецептов в стиле TikTok. Пользователи могут скроллить рецепты вертикально, просматривать доступные ингредиенты и выбирать блюда для приготовления.

## Основные возможности

### 🎯 Вертикальный скроллинг
- **Свайп вверх/вниз** для навигации между рецептами
- **Плавные анимации** перехода между карточками
- **Адаптивный дизайн** для мобильных устройств

### 🍽️ Умная фильтрация рецептов
- **Анализ ингредиентов** из холодильника пользователя
- **Множественные фильтры** для разных типов рецептов
- **Индикатор доступности** с процентным соотношением
- **Статистика по категориям** рецептов

#### Типы фильтров
- **Все рецепты** - показ всех доступных рецептов
- **Полностью доступные** - рецепты со 100% ингредиентов
- **Частично доступные** - рецепты с 50-99% ингредиентов
- **Недостающие ингредиенты** - рецепты с менее 50% ингредиентов

### 🎨 Интерактивные элементы

#### Кнопки действий
- **❤️ Лайк** - добавление в избранное
- **🔖 Сохранить** - сохранение рецепта
- **📤 Поделиться** - отправка рецепта

#### Управление навигацией
- **↑ Предыдущий** - переход к предыдущему рецепту
- **🔄 Заменить** - случайная замена текущего рецепта
- **↓ Следующий** - переход к следующему рецепту

#### Выбор рецепта
- **"Выбрать рецепт"** - основная кнопка выбора
- **"Подробнее"** - переход к детальному просмотру

### 📊 Информационные панели

#### Панель ингредиентов
- **Показать/скрыть** список ингредиентов
- **Цветовая индикация** доступности (зеленый/красный)
- **Список недостающих** ингредиентов для покупки

#### Статистика рецепта
- **Время приготовления** (подготовка + готовка)
- **Количество порций**
- **Рейтинг** и отзывы
- **Информация об авторе**

### 🎖️ Система достижений

#### Типы достижений
- **🏆 Трофей** - за особые заслуги
- **⭐ Звезда** - за активность
- **🔥 Огонь** - за скорость и эффективность

#### Прогресс достижений
- **Отслеживание прогресса** для заблокированных достижений
- **Мгновенное уведомление** при разблокировке
- **Детальная статистика** просмотров

### 📱 Мобильная оптимизация

#### Жесты
- **Свайп вверх** - следующий рецепт
- **Свайп вниз** - предыдущий рецепт
- **Долгое нажатие** - дополнительные опции

#### Адаптивный интерфейс
- **Полноэкранный режим** на мобильных устройствах
- **Оптимизированные размеры** кнопок для касания
- **Улучшенная читаемость** текста

## Компоненты

### DiscoverPage
Основная страница раздела с логикой навигации и управления состоянием.

### DiscoverCard
Карточка рецепта с полной информацией и интерактивными элементами.

### DiscoverCardSkeleton
Скелетон для отображения во время загрузки данных.

### EmptyDiscoverCard
Компонент для пустого состояния с подсказками.

### SwipeIndicator
Индикатор направления свайпа для навигации.

### ProgressIndicator
Индикатор прогресса просмотра рецептов.

### FilterSettings
Настройки фильтрации с выпадающим меню выбора типа рецептов.

### FilterInfo
Информация о текущем фильтре с количеством рецептов.

### FilterStats
Статистика по всем категориям рецептов с визуальными индикаторами.

### QuickFilterTabs
Быстрые вкладки для переключения между фильтрами.

### RecipeSelectedNotification
Уведомление о выборе рецепта.

### GestureHint
Подсказка по жестам для новых пользователей.

### ViewStats
Статистика просмотра и активности пользователя.

### AchievementBadge
Система достижений с прогрессом и наградами.

## Технические особенности

### Производительность
- **Ленивая загрузка** изображений
- **Оптимизированные анимации** с CSS transforms
- **Эффективное управление состоянием** с React hooks

### Доступность
- **ARIA-метки** для скринридеров
- **Клавиатурная навигация** для всех элементов
- **Высокий контраст** для лучшей читаемости

### Анимации
- **Плавные переходы** между состояниями
- **Микроанимации** для обратной связи
- **Оптимизированные** для 60fps

## Интеграция

### Redux Store
- **fridgeSlice** - данные холодильника
- **recipesSlice** - данные рецептов
- **uiSlice** - настройки интерфейса

### API
- **recipesService** - загрузка рецептов
- **fridgeService** - данные ингредиентов
- **favoritesService** - управление избранным
