# Тестирование формата JSON для API

## 🧪 Как проверить правильность JSON

### 1. Через DevTools API Logger

1. Откройте приложение: http://localhost:5174/refook_v3/
2. Откройте DevTools панель в правом верхнем углу
3. Перейдите в раздел "API Logger"
4. Создайте новый рецепт с шагами
5. Посмотрите JSON в API Logger

### 2. Через консоль браузера

1. Откройте DevTools (F12)
2. Перейдите на вкладку Console
3. Создайте рецепт - увидите логи:
   - `📥 Данные из формы:` - что приходит из формы
   - `📤 Отправляемые данные на API:` - что отправляется на сервер

### 3. Тестовая страница

Откройте: http://localhost:5174/refook_v3/src/test-recipe-json.html

## ✅ Ожидаемый формат JSON

```json
{
  "name": "Название рецепта",
  "description": "Описание рецепта",
  "kitchen": "RUSSIAN",
  "level": "EASY",
  "cookTime": 1800,
  "allTime": 3600,
  "portion": 4,
  "photos": [
    "12fcd2f5-f976-44bc-bd33-c495a8daff34",
    "a3bd2a15-dc26-43a7-ae92-f7c62b9edc21"
  ],
  "tags": null,
  "ingredients": [
    {
      "id": "fc01f44c-cc90-44c9-b0f4-53f586f41e30",
      "count": 500,
      "measure": "GR"
    }
  ],
  "steps": [
    {
      "index": 1,
      "name": "Выбор картошки",
      "description": "Привет, меня зовут описание говна",
      "photos": [
        "12fcd2f5-f976-44bc-bd33-c495a8daff34",
        "a3bd2a15-dc26-43a7-ae92-f7c62b9edc21"
      ],
      "ingredients": [
        {
          "id": "fc01f44c-cc90-44c9-b0f4-53f586f41e30",
          "count": 1,
          "measure": "KG"
        }
      ],
      "time": 0
    }
  ]
}
```

## 🔍 Что проверить

### Обязательные поля:
- ✅ `name` - название рецепта
- ✅ `description` - описание рецепта  
- ✅ `level` - сложность (EASY/MEDIUM/HARD)
- ✅ `steps[].index` - индекс шага
- ✅ `steps[].description` - описание шага

### Опциональные поля:
- ✅ `kitchen` - тип кухни (RUSSIAN/ASIAN)
- ✅ `cookTime` - время готовки в секундах
- ✅ `allTime` - общее время в секундах
- ✅ `portion` - количество порций
- ✅ `photos` - массив ID фотографий
- ✅ `tags` - временно отправляется как `null`
- ✅ `ingredients` - массив ингредиентов
- ✅ `steps[].name` - название шага
- ✅ `steps[].photos` - фотографии шага
- ✅ `steps[].ingredients` - ингредиенты шага
- ✅ `steps[].time` - время шага в секундах

## 🚨 Частые ошибки

1. **Пустое название шага** - должно быть `undefined`, а не пустая строка
2. **Неправильный тип данных** - время должно быть числом, а не строкой
3. **Отсутствие обязательных полей** - `name`, `description`, `level`
4. **Неправильный формат ингредиентов** - должен содержать `id`, `count`, `measure`

## 🛠 Отладка

Если JSON неправильный:

1. Проверьте консоль браузера на ошибки
2. Убедитесь, что все поля заполнены корректно
3. Проверьте типы данных (числа vs строки)
4. Убедитесь, что массивы не содержат `null` или `undefined`
